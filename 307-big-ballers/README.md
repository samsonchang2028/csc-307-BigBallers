# OptiCart

Your groceries, one destination.

## Getting Started

### How to Sign Up

1. Go to the app and click **Sign up** on the login page (or navigate to `/login`).
2. Enter your **email address** and a **password**.
3. Click the **Sign up** button.
4. Check your email inbox for a confirmation link and click it to verify your account.
5. Return to the login page, enter your credentials, and click **Log in**.

> If you already have an account, click **Log in** and enter your email and password directly.

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
